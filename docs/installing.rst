Installing AxterDB
=================

This is the documentation for AxterDB, a library for Python to help with interacting with AxterDB.

Prerequisites
-------------
Pycord works with Python 3.8 or higher. Support for earlier versions of Python
is not provided. Python 2.7 or lower is not supported. Python 3.7 or lower is not supported.

.. _installing:

Installing
----------

You can get the library directly from PyPI: ::

    python3 -m pip install -U axterdb

If you are using Windows, then the following should be used instead: ::

    py -3 -m pip install -U axterdb
